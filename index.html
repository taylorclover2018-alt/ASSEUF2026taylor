Transparência do Transporte Universitário

Este sistema foi criado para informar de forma transparente os valores das rotas: Sete Lagoas e Curvelo.

A divisão do auxílio é feita de forma imparcial, baseada:

  • Valor real das diárias
  • Número de dias rodados
  • Regra 70% para quem rodou no dia
  • 30% para quem não rodou
