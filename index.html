Transparência do Transporte Universitário

Este sistema foi criado para informar de forma clara e imparcial os valores das rotas de Sete Lagoas e Curvelo.

A divisão do auxílio considera:

  • Valor real das diárias
  • Número de dias rodados
  • Regra 70% para quem rodou sozinho
  • 30% para quem não rodou naquele dia
  • 50% para cada quando ambas rodaram
